# ☕ The Coffee Lab: A Quadratic Strategy Sim

**The Coffee Lab** is a management simulation designed to make quadratic functions and parabola analysis intuitive and interactive. Built with vanilla JavaScript, it challenges players to run an independent coffee shop and out-manage a corporate rival using mathematical optimization.

---

## 🎯 Educational Objectives
Designed for Algebra and Pre-Calculus students to practice:
* **Vertex Identification:** Finding the maximum "Taste Score" for temperature, sweetness, and intensity.
* **Parabola Forms:** Analyzing data points in Symmetry (Midpoint), Standard Form ($x = -b/2a$), and Vertex Form.
* **Quadratic Optimization:** Balancing variables to maximize profit while minimizing "walkaways."

---

## 🕹️ How to Play
1. **Analyze the Forecast:** Different weather types (Sunny, Heatwave, Rainy) shift the ideal coffee recipe.
2. **Optimize the Recipe:** Adjust sliders for Temperature, Sweetness, and Intensity. Each is governed by a hidden quadratic equation—find the vertex for a perfect score.
3. **Set the Price:** Balance high margins with customer conversion rates to maximize daily profit.
4. **Outperform Roaster Co.:** Reach **$15,000** first to win the game and secure the building.

---

## 🛠️ Technical Features
* **Zero External Dependencies:** 100% Vanilla JS, CSS3, and HTML5. No external font fetches or libraries (Coolmath Games compliant).
* **Responsive Design:** Fluid layout that adapts to Desktop, Chromebooks, and Tablets.
* **Privacy Focused:** Fully COPPA/GDPR compliant. No user data collection or external tracking.
* **Procedural Math:** Every new game generates unique quadratic coefficients for high replayability.

## 👨‍🏫 Note for Educators
The "Advanced Research Lab" in-game is designed to bridge the gap between "guessing" and "calculating." Students start with trial-and-error using secant tables and eventually "purchase" the ability to see equations in Standard or Vertex form to find the exact coordinates of the peak.
